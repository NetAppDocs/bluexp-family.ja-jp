BlueXPのマニュアル

NetApp BlueXPは、オンプレミス環境とクラウド環境にわたってデータを構築、保護、ガバナンスするための単一のコントロールプレーンです。製品別のドキュメント領域を参照するか、上記の検索を使用してBlueXPドキュメント内の任意の場所にコンテンツを検索します。

すべて
セットアップと管理

BlueXPの使用を開始し、コネクタやアカウントなどを管理する方法について説明します。

リリースノート

BlueXPのすべての新機能について説明します。

NetApp ONTAP 対応の Amazon FSX

AWS が管理する標準サービスとして、 ONTAP の全機能を使用できます。

Amazon S3ストレージ

AWSアカウント内のAmazon S3バケットを検出して管理します。

API

クラウドベースのリソースをプログラムで管理できます。

自動化カタログ

スクリプト化されたソリューションを使用して、NetApp製品とサービスの導入と統合を自動化できます。

Azure BLOBストレージ

AzureサブスクリプションでAzure BLOBストレージを検出および管理します。

Azure NetApp Files の特長

コアビジネスアプリケーション向けのハイパフォーマンスなファイルストレージサービス。

バックアップとリカバリ

クラウドとオンプレミスのデータをバックアップ、リストア

分類

アプリケーションデータとクラウド環境のプライバシーを確保

クラウド運用

クラウドコンピューティング関連の支出を可視化し、削減可能なコストを特定できます。

Cloud Volumes ONTAP

クラウドストレージを低コストでインテリジェントに、コンプライアンスに準拠し、セキュアに構築できます。

Cloud Volumes Service for Google Cloud

ファイルストレージをセキュアに、保護、高速化します。

コピーと同期

オンプレミスとクラウドのデータストア間でデータを同期

デジタルアドバイザ（Active IQ ）

実用的な情報でストレージの健常性を最適化

デジタルウォレット

ライセンスとサブスクリプションを管理および監視します。

ディザスタリカバリ

ディザスタリカバリサイトとしてVMware Cloud on Amazon FSx for ONTAPを使用して、オンプレミスのVMwareワークロードを保護します。

Eシリーズシステム

ハイブリッドマルチクラウド全体のデータを包括的に把握できるEシリーズシステムをご紹介します。

経済効率

現在または予測されている低容量のクラスタを特定し、データ階層化または容量追加の推奨事項を実装します。

エッジキャッシュ

分散した従業員にリアルタイムのグローバルファイル共有を提供します。

Google クラウドストレージ

GoogleアカウントでCloud Storageバケットを検出して管理します。

Kubernetes クラスタ

ハイブリッドクラウド内の Kubernetes クラスタの高度なデータ管理。

移行レポート

ストレージ環境のファイル数、ファイルシステムツリーの深さと幅、大容量ファイルなどを特定します。

オンプレミスの ONTAP クラスタ

System ManagerとBlueXPのデータサービスを使用して、AFF システムとFAS システムをBlueXPから管理できます。

運用の耐障害性

ファームウェアとソフトウェア設定のリスクを自動で修正し、ONTAP クラスタの健全性を維持

ランサムウェアからの保護

ランサムウェア攻撃につながる可能性のある異常を検出ワークロードの保護とリカバリ

修正

BlueXPのアクションを自動化して、プロビジョニングを高速化し、手動作業を排除します。

レプリケーション

バックアップとディザスタリカバリをサポートするために、ストレージシステム間でデータをレプリケートします。

StorageGRID

StorageGRID システムの検出と管理

Sustainability Dashboard

ストレージシステムの持続可能性を分析します。

階層化

オンプレミスの ONTAP ストレージをクラウドに拡張

ボリュームキャッシュ

書き込み可能なキャッシュボリュームを作成して、データへのアクセスを高速化したり、アクセス頻度の高いボリュームのトラフィックをオフロードしたりします。
